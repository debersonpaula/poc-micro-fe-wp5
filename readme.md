# Micro Frontend WP5

This sample was designed to be a boilerplate for micro-frontend architecture, by using Webpack 5 and ModuleFederationPlugin.

---

## Module: __mod-app__

Application encapsulation, provides theming, data and routing for all modules.

### Requires
- library
- mod-header

---
